Основной проект состоит из rest Api для управлением базы данных на тему Цветочный магазин, и Клиентской части
в этом проекте научился разрабатывать свой rest Api и клиентскую часть для взаимодействие с Api, разработал базу данных для цветочного магазина на 15 таблиц,
писал свой docker и docker compose- для сборки и развертывание проекта на удаленном сервере
